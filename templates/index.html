WEB3: Employee Recognition

An online application for managing and creating employee recognition awards
James Mills, Nathaniel Piche, Louise McGuire
Oregon State University | Fall 2016

There are two sections of the website with separate access privileges: Admin and Awards. If you are granted access as an Award Creator, you will be able to create new awards and view/delete awards previously granted. Admin users are able to add/edit/delete both Award Creators and other Admin users, as well as generate reports on user activity.

For users

To use this tool, you must register and create a new account. After registering for an account, you will be able to create new awards, view/delete awards previously granted, and edit your user profile.

To begin, click on the register button below.


If you've already registered, click log in.

For admin

If you've got admin access to our website, simply follow the link below to log in. Privileges are granted on a case-by-case basis. If you don't already have it, you probably don't need it.
