WEB3: Employee Recognition

An online application for managing and creating employee recognition awards.


Register

To use this tool, you must register and create a new account. After registering for an account, you will be be able to create new awards, view/delete awards previously granted, and edit your user profile. To begin, click on the register button below.

Login

There are two sections of the website with separate access privileges: Admin and Awards. If you are granted access as an Award Creator, you will be able to create new awards and view/delete awards previously granted.

If you are granted access as an Admin, you will be able to add/edit/delete Award Creators, add/edit/delete other Admin users, and perform Business Intelligence reporting operations.
